# PWA インストールテスト手順

## 🖥️ PC版Chrome（現在の環境）

### アドレスバーを確認
1. ブラウザのアドレスバー（URLの右側）を確認してください
2. **⊕ アイコン**または**インストールアイコン**が表示されているか確認
3. 表示されていれば、それをクリック → インストールダイアログが表示されます

### デバッグパネルでテスト
1. 右側のデバッグパネルの **"Trigger Install"** ボタンをクリック
2. インストールダイアログが表示されるか確認
3. 表示されない場合は **"Show Manual Install"** をクリック

### Chrome メニューから確認
1. Chrome の メニュー（⋮）を開く
2. **"「愛車運勢診断」をインストール..."** というメニュー項目があるか確認
3. あればクリック → インストールダイアログが表示されます

## 📱 Android Chrome（推奨テスト環境）

### スマートフォンでテスト
1. Androidスマートフォンで Chrome ブラウザを開く
2. `https://car-unsei.jp` にアクセス
3. 30秒ほど待つ
4. **インストールプロンプトが自動的にポップアップ表示される**

### 手動インストール（Android）
1. Chrome メニュー（⋮）を開く
2. **"アプリをインストール"** をタップ
3. インストールダイアログが表示される

---

## 🔍 なぜPC版では自動ポップアップが表示されないのか？

### Chrome の仕様
- **Android Chrome**: エンゲージメント要件を満たすと、画面下部に大きなポップアップが表示される
- **PC Chrome**: アドレスバーに小さなアイコンが**静かに**表示される（控えめ）

これはChromeの仕様で、PC版では邪魔にならないように、自動ポップアップではなくアドレスバーのアイコン表示になっています。

## 💡 ヒント

### PC版Chromeでインストールプロンプトを確認したい場合

**Chrome Flags を有効化:**
```
chrome://flags/#bypass-app-banner-engagement-checks
```
→ "Enabled" に設定 → Chrome を再起動

これで、PC版でも自動ポップアップが表示されるようになります（テスト用）。

ただし、**実際のエンドユーザーはこの設定をしていない**ので、本番環境では：
- PC: アドレスバーのアイコン
- Android: 自動ポップアップ

という動作になります。
