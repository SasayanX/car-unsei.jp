# キャッシュ完全クリア手順

## 🚨 重要: Service Workerとキャッシュを完全にクリアする

### 手順1: Chrome DevTools でService Workerをアンインストール

1. **Chrome DevTools を開く**
   ```
   F12 または Ctrl + Shift + I
   ```

2. **Application タブを開く**

3. **左メニューから "Service Workers" を選択**

4. **"Unregister" ボタンをクリック**
   - すべてのService Workerを削除

5. **左メニューから "Storage" を選択**

6. **"Clear site data" ボタンをクリック**
   - すべてのキャッシュ、Cookie、Storageを削除

### 手順2: ブラウザのキャッシュをクリア

1. **Chrome の設定を開く**
   ```
   Ctrl + Shift + Delete
   ```

2. **以下を選択:**
   - ✅ キャッシュされた画像とファイル
   - ✅ Cookie と他のサイトデータ
   - 期間: **全期間**

3. **"データを削除" をクリック**

### 手順3: ハードリロード

1. **ページを開く**
   ```
   https://car-unsei.jp
   ```

2. **ハードリロードを実行**
   ```
   Ctrl + Shift + R
   または
   Ctrl + F5
   ```

3. **5秒待つ**

4. **新しいモーダルが表示される！**
